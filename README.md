# 📊 Customer Data Analysis (SQL)

## 📌 Project Overview
This project analyses customer data to identify trends in revenue, customer behaviour, and subscription patterns using SQL.

---

## 🛠 Tools & Technologies
- SQL (MySQL)

---

## 📁 Dataset
Sample customer dataset containing demographic and subscription information.

---

## 🔍 Key Analysis
- Revenue by region.  
- Average spend by subscription type.  
- Identification of high-value customers. 
- Customer tenure and retention analysis. 

---

## 📈 Key Insights
- The South region generates the highest revenue, indicating strong customer spending and market potential in that region.
- Customers on the Premium subscription plan have the highest average spend, suggesting this segment contributes significantly to revenue.
- A small group of customers contributes disproportionately to revenue, highlighting the importance of high-value customers.
- Regions with higher average tenure indicate stronger customer loyalty and retention.

---

## 📌 Outcome
This project demonstrates SQL skills in data querying, aggregation, and business analysis.
